# Cursor Custom Prompts (copy these into Cursor → Settings → Custom Prompts)

## /plan-small
"Outline a 3–5 step plan to implement the change. Keep steps tiny, token-light. No refactors."

## /implement-unit
"Implement only the minimal code to satisfy the spec I give next. Single-file diff. No extra comments."

## /tests-min
"Generate the smallest test set that covers: happy path, one edge, one error. Keep changes under 30 lines."

## /doc-1liner
"Add one-line docstrings to public functions/classes only."

## /push
"Remind me to run Task: 'Git Push + Log' and suggest a short commit message ≤8 words."
